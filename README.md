# Mail-sender
Send mail through python using smtplib and getpass.
# Requirments
1. Python (v3.0 or above).
2. import smtplib
3. import getpass

# Flow of code
1. Input sender email id.
2. Input password.
3. Input recipient email id.

# Voila! your message sent->:)
